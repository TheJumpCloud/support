#### Name

Linux - Install Snap Daemon | v1.0 JCCG

#### commandType

linux

#### Command

```
snap install slack --classic
```

#### Description

Install Slack app with the Snap app manager. To learn more about Snap, visit [SnapCraft's website](https://snapcraft.io/)
